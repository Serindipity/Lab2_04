1. What is .gitignore? You can write the answer either in Korean or English.
.gitignore 파일이란 git으로 프로젝트를 관리할 때, 원하지 않는 Backup File이나 Log File , 혹은 컴파일 된 파일들 등
관리할 필요가 없는 특정 파일들을 git 버전 관리에서 제외할 수 있는 설정 파일이다. 
2. Why do Github users need an SSH key pair? (Users can use either SSH key pair or github account info.) Please write down a brief explanation of SSH key.
SSH key는 pair로 이루어져 있다. 하나는 GitHub 등의 서비스와 공유되는 public key, 하나는 오직 내 컴퓨터에만 저장되는 private key이다.  
key들이 매칭될 때 접근이 가능하다. SSH key pair는 public key에서 private key를 역설계하지 못하도록 한다.
